# Blog API

API backend simple de gestion d'articles de blog développée avec Node.js, Express, SQLite et Swagger.

## Fonctionnalités

- Créer un article
- Lire tous les articles
- Lire un article par ID
- Modifier un article
- Supprimer un article
- Rechercher un article
- Filtrer les articles par catégorie, auteur ou date
- Documentation Swagger

## Technologies utilisées

- Node.js
- Express.js
- SQLite
- Swagger UI
